# rye-gioui

Gio UI library with Rye language

## Build

```sh
# runs go generate and build into the local .bin folder
make all

## Runs an example
make run
```

## Usage


```sh
## Run the Hellp example
rye-giou ./examples/hello_gio.rye

## Run the Click example
rye-giou ./examples/click_counter.rye
```
